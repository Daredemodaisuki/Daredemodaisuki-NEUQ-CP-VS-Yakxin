1.	计算日期差
利用下面的结构编写一个程序用来计算两个日期之间相差的天数：
1.	struct Date {
2.	   int year;
3.	   int month;
4.	   int day;5.	};
①	输入形式
输入两个日期，每个日期分占一行，在一行中日期的年、月、日是三个整数，以空格分隔。并假设第二个日期大于或等于第一个日期；
②	输出形式
第二个日期与第一个日期间相差的天数。
2.	电话薄排序*
编写一个程序，输入N个用户的姓名和电话号码，按照用户姓名的词典顺序排列输出用户的姓名和电话号码。
①	输入形式
用户首先在第一行输入一个正整数，该正整数表示待排序的用户数目，然后在下面多行输入多个用户的信息，每行的输入格式为“姓名 电话”，输入姓名时将姓和名一起输入，中间无空格，以回车结束每个用户的输入；
②	输出形式
程序输出按姓名词典排序排序后的结果，每行亦按照“姓名 电话”格式输出，姓名和电话字段中间没有空格，要求用户姓名不能超过10个字符，超出10个字符时候只取前10个字符作为姓名。电话号码不能超过10位，超过10位时只按10位处理。输出姓名、电话字段各占12个字符宽，输出格式采用默认对齐方式。另外，用户的数量要求不超过50个。
